Terminal Management System(TMS)

This project, built with Node.js and Moleculer.js, focuses on analyzing and controlling set-top boxes. It captures and processes logs including application usage, CPU data, and RAM availability, which are stored in Elasticsearch. The system performs queries and aggregations on the data to analyze individual devices and the entire set. The project also tracks device status through regular heartbeat signals, recorded in MySQL and InfluxDB. InfluxDB specifically helps determine the number of online devices within specified date ranges. Additional features include firmware updates for Android devices. Authorized personnel create update files stored in Minio servers. Upon device restart, updates are checked, installed, and logged in Elasticsearch. In summary, this compact and efficient project uses Node.js and Moleculer.js to analyze and control set-top boxes. It collects and processes logs, tracks device status, and enables firmware updates, ensuring optimal performance and management of the devices.

Nepal Telecommunications Authority(NTA)

This project, developed with PHP and Laravel under the guidance of my superiors, aims to safeguard internet users from potentially harmful or inappropriate websites. Its primary objective is to create a platform where users can submit domain names that should be blocked for the protection of others. My involvement in the project focused on implementing key components. Firstly, I developed an API that allows the public to register valid domains and categorize them accordingly. This enables users to contribute to the identification and filtering process. Additionally, the project includes a verification system for authorized personnel. They have the responsibility to review and filter through the registered domains, ensuring their accuracy and appropriateness. In summary, this compact and elegant solution utilizes PHP and Laravel to empower internet users to actively participate in identifying and blocking undesirable websites. Through the registration API and the verification process, we aim to create a safer online environment for all users.
